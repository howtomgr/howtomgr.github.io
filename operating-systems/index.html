Raspberry Pi OS Installation Guide

Raspberry Pi OS (formerly Raspbian) is a free and open-source Debian-based operating system optimized for Raspberry Pi hardware. It provides a complete desktop environment and serves as a FOSS alternative to proprietary embedded operating systems, ideal for IoT, education, and hobbyist projects.
